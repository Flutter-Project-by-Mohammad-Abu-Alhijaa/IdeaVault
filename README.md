# IdeaVault

</div>

### Your Localized Flutter Note-Taking Experience

IdeaVault is a lightweight, intuitive, and fully localized note-taking application built with Flutter. It enables users to create, manage, and delete personal notes while providing culturally adaptable UI through multilanguage support. The app uses SQFlite for local data persistence and follows a clean, scalable MVC structure.
